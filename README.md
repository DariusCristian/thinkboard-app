ThinkBoard is a full-stack notes application built with the MERN stack (MongoDB, Express, React, Node.js). It includes a REST API for note management, MongoDB persistence, rate limiting middleware, and a responsive React UI.

---

## Features

- Create notes
- View note details
- Delete notes
- REST API built with Express
- MongoDB integration (Mongoose)
- Middleware setup + rate limiting
- Responsive React UI (Vite)

---

## Tech Stack

**Backend**
- Node.js
- Express.js
- MongoDB + Mongoose
- Rate limiting middleware

**Frontend**
- React
- Vite

---

## Requirements

- Node.js (LTS recommended)
- npm
- MongoDB (local or MongoDB Atlas)
